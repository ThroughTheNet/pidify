Pidify

Synopsis

A Ruby module to simplify storing and deleting the PID of a running program. It also provides the ability to kill a running program whose PID it has already saved. This allows a program to check if there is currently another running instance of itself, and give it the ability to kill that instance based on PID.

Note that this does no special process checking and relies solely on the PID files it creates and maintains.

See Pidify for more information and examples.
